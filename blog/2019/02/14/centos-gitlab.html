      # CentOS安装Gitlab

      发布于 02/14/2019, 11:54 PM
      分类: 运维 标签: Gitlab

      • 安装
      • 配置并启动

      # 安装

      curl https://packages.gitlab.com/install/repositories/gitlab/gitlab-ce/script.rpm.sh | sudo bash
yum install gitlab-ce
      1
      2

      因为下载太慢了所以手动下载传到服务器上面去 https://packages.gitlab.com/gitlab/gitlab-ce/packages/el/7/gitlab-ce-11.7.5-ce.0.el7.x86_64.rpm 或者 https://mirrors.tuna.tsinghua.edu.cn/gitlab-ce/yum/el7/gitlab-ce-11.7.5-ce.0.el7.x86_64.rpm 然后

      yum localinstall gitlab-ce-11.7.5-ce.0.el7.x86_64.rpm
      1

      # 配置并启动

      gitlab-ctl reconfigure
      1

      安装完成后打开80端口，显示启动成功
