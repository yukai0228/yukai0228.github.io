        # 安装helm

        离线安装方式 要求环境

        • docker
        • docker-compose
        • python
        • openssl

        下载离线安装包

        wget https://storage.googleapis.com/harbor-releases/release-1.7.0/harbor-offline-installer-v1.7.5.tgz
tar -zxvf harbor-offline-installer-v1.7.5.tgz 
# 编辑配置文件
vim harbor.cfg
## 编辑里面的hostname 
./prepare
./install.sh
